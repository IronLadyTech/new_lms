1. User (Learner) flow

Sign in → learner app → courses, dashboard, calendar, support, profile

flowchart TD
  A[Visit /auth/login] --> B{Signed in?}
  B -->|No| C[Login / Sign up / Google / Guest]
  C --> D{Account type}
  D -->|Guest| E[Preview only - locked panels]
  D -->|Real user| F[Load Firestore profile]
  F --> G{Blocked?}
  G -->|Yes| H[Account blocked screen]
  G -->|No| I[Learner app /app/*]
  B -->|Yes student| I
  B -->|Yes staff| J[/portal - Admin or User view]

  I --> K[Bottom nav + header]
  K --> L[/app/dashboard - Home]
  K --> M[/app/calendar]
  K --> N[/app/progress]
  K --> O[/app/support]
  K --> P[/app/profile]
  I --> Q[/app/home - Courses]
  Q --> R[Enroll / Continue]
  R --> S[/app/course/:id - Resources]
  L --> T[Announcements streak events]

2. Admin (Staff) flow

Staff login → portal → admin panel → manage LMS

flowchart TD
  A[Staff signs in] --> B[/portal]
  B --> C{Choose view}
  C -->|Open admin| D{Role?}
  C -->|See as user| E[/app/home learner view]
  D -->|Moderator / Admin| F[/admin]
  D -->|Super Admin| G[/superadmin]
  F --> H[Admin sidebar tabs]
  G --> H
  H --> I[Overview stats charts]
  H --> J[Users roles block]
  H --> K[Support tickets]
  H --> L[Progress + modal]
  H --> M[Activity log]
  H --> N[Calendar events]
  H --> O[Announcements]
  H --> P[Courses CRUD]
  H --> Q[Resources lock/upload]
  H --> R[Batches groups]
  H --> S[View as user / Sign out]
  E --> T[Header Admin link to portal]
